# Electricity Billing System

Full Stack Electricity Billing System built using Java Spring Boot and React.

## Features

* Add customer details
* Generate electricity bill
* Update customer records
* Delete customer records
* REST API integration
* Responsive frontend UI
* Full CRUD operations

## Tech Stack

### Frontend

* React.js
* Axios
* CSS

### Backend

* Java
* Spring Boot
* Maven

### Database

* MySQL

## Project Structure

```bash
electricity-billing-system/
│
├── backend/
├── frontend/
├── database/
└── README.md
```

## Backend Setup

1. Open backend folder

```bash
cd backend
```

2. Run Spring Boot application

```bash
mvn spring-boot:run
```

Backend runs on:

```bash
http://localhost:8080
```

## Frontend Setup

1. Open frontend folder

```bash
cd frontend
```

2. Install dependencies

```bash
npm install
```

3. Start React application

```bash
npm start
```

Frontend runs on:

```bash
http://localhost:3000
```

## Database Setup

1. Open MySQL
2. Create database
3. Run schema.sql file from database folder

## API Endpoints

### Get Customers

```bash
GET /api/customers
```

### Add Customer

```bash
POST /api/customers/add
```
